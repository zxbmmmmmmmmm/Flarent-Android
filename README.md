# Flarent

Flarent is a native Flarum Android client built with Kotlin and Jetpack Compose.

The project can be used as a starting point for building a branded mobile client for a single Flarum community. 

## Features

- Browse discussions
- View posts
- Read notifications
- Reply to discussions and posts with a Markdown editor and preview
- ...

## Supported extensions

The application has built-in support for the following extensions:

- [fof/gamification](https://github.com/FriendsOfFlarum/gamification)
- [fof/reactions](https://github.com/FriendsOfFlarum/reactions)
- [fof/upload](https://github.com/FriendsOfFlarum/upload)

Features of the following extensions are partially implemented:

- [fof/best-answer](https://github.com/FriendsOfFlarum/best-answer)
- [fof/user-bio](https://github.com/FriendsOfFlarum/user-bio)
- [fof/forum-stats-widget](https://github.com/FriendsOfFlarum/forum-stats-widget)
- [fof/prevent-necrobumping](https://github.com/FriendsOfFlarum/prevent-necrobumping)

## Build & Rebrand

See [docs/BUILDING.md](docs/BUILDING.md) for the full build guide, including:

- replacing launcher icons;
- adding `google-services.json`;
- building debug and release artifacts.
